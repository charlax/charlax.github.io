Resources to learn coding

  • Mon 09 September 2013
  • code

I'm often asked about resources to learn programming. Here are a few of them.

  • charlax/professional-programming: a collection of full-stack resources for programmers
  • Coursera (the best IMO)
  • Udacity
  • Udemy
  • Khan Academy
  • Learn Python the Hard Way
  • CodeAcademy
  • iTunesU has some good courses.

There are numerous of them, and this does not even count the books. There are also numerous articles about the best resources to learn coding, and the best method of learning.

Here's my only advice: nothing beats learning by doing. Try to build something meaningful and exciting to apply what you learn.
